# SS14 КЗ Калькулятор

Браузерный инструмент для расчета сроков наказания по Корпоративному Закону (КЗ) Space Station 14. Быстро добавляйте статьи, выбирайте модификаторы и смотрите итоговый срок в реальном времени.

Сделан на HTML, CSS и JavaScript без зависимостей. Все данные хранятся локально в браузере.

Репозиторий: https://github.com/wl-kz/calculator

## Возможности

- Выбор статей из полной матрицы КЗ (7 типов, 5 уровней)
- Быстрый поиск по коду, названию или описанию
- Модификаторы: умысел, завершённость, готовность, намерения
- Подсчёт добавочных минут за дополнительные статьи
- Расчёт с учётом повторений и уровневых изменений
- Сохранение дела в браузере
- Адаптивный дизайн (всё работает на мобильных)
- Две компоновки: раздельная и снизу

## Файлы

- index.html - основное приложение
- style.css - оформление
- app.js - логика расчётов и интерфейса
- data/articles.js - описание всех статей КЗ
- assets/ - фоновые текстуры из Wiki SS14

## Запуск

Откройте index.html в браузере или разместите на GitHub Pages:

1. Settings -> Pages
2. Deploy from branch -> main, folder /(root)

Справочная информация основана на редакции КЗ Corvax Whitelist.
