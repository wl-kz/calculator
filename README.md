# Калькулятор срока КЗ

Статическая страница без сборки и зависимостей. Откройте `index.html` локально или опубликуйте каталог через GitHub Pages.

## GitHub Pages

1. Создайте репозиторий и загрузите `index.html`, `style.css`, `app.js`, каталоги `data` и `assets`.
2. Откройте **Settings → Pages**.
3. В **Build and deployment** выберите **Deploy from a branch**, ветку `main`, каталог `/ (root)`.

Расчёт основан на приложенной редакции КЗ: диапазоны от 15 до 25 и от 30 до 45 минут, правила сложения статей и уровневые модификаторы. Это справочный инструмент, а не замена действующей редакции КЗ.

Данные статей отделены от логики приложения и находятся в `data/articles.js`.

Звёздный фон основан на двух фоновых текстурах Space Station 14 Вики: `assets/stars-bg-1.png` и `assets/stars-bg-2.png`.
